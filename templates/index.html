FSMForge

FSMForge is an automated counter design platform that bridges the gap between theoretical sequential logic design and practical hardware implementation. The tool accepts a custom state sequence and flip-flop type, automatically derives excitation equations, generates gate-level schematics, and produces synthesizable Verilog along with a testbench.

Key Capabilities

A comprehensive academic and industry-oriented automation platform for finite state machine based counter design.

Automated FSM Synthesis

Converts user-defined state sequences into minimized Boolean excitation equations using symbolic logic techniques.

Multi Flip-Flop Support

Supports D, T, JK, and SR flip-flop based counter designs with accurate excitation modeling.

Gate-Level Visualization

Automatically generates readable and interpretable gate-level diagrams for each excitation function.

HDL & Testbench Generation

Produces synthesizable Verilog modules and simulation-ready testbenches suitable for academic and industry workflows.
