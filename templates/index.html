Counter Design Automation Tool

An end-to-end digital design framework developed to automate the synthesis of synchronous counters and finite state machines (FSMs) from user-defined state sequences. The tool translates high-level behavioral specifications into optimized Boolean equations, gate-level schematics, and synthesizable Verilog code, significantly reducing manual design effort and human error.

Project Overview

Conventional counter and FSM design workflows require extensive manual derivation of state tables, excitation equations, Boolean minimization, and HDL coding. This project addresses these limitations by providing a systematic, automated, and error-resilient design flow suitable for both academic learning and professional VLSI environments.

The framework supports arbitrary state sequences, multiple flip-flop types (D, T, JK, SR), automated visualization of gate-level logic, and generation of synthesizable Verilog modules along with testbenches compatible with standard EDA tools.
